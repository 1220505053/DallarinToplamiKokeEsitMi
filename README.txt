C# dilinde size 2 düğümden oluşan ikili bir ağacın kökü verildiğinde, bu değerin alt düğümlerin(child) toplamı olup olmadığını bulan programdir. Binarytreelerin alt dallari koke esitligini tespit etmekte kullanilabilir.


Indirdigimiz dosyalari calistirip ustunde degisiklikler yapmak icin C# diline entegre bir uygulamaya ihtiyacimiz var. Benim onerim Visual Studio olacak.
Visual Studio' yu ucretsiz bir sekilde Micrososft' un web sitesinden indirebilirsiniz.

Visual Studio 2022 indirmek icin asagidaki linki kullanabilirsiniz:
https://visualstudio.microsoft.com/downloads/

 Visual Studio kurulumu bittikten sonra .cs dosyalarina su sirayla erisebilirisniz:
 1- Visual Studio' yu acin ve "File -> New -> Project" sirasini takip edin
 2- "Console App" basligini listeden secin ve isimlendirin.
 3- Proje olusturulduktan sonra "Program.cs" seklinde isimlendirildigini goreceksiniz. Bu kutu altinda yazilan kodlarin islevsel bir sekilde
 bir arada durmasini sagliyor.
 4- Sayfanin en ust kismindaki Visual Studio ikonunun yanindaki "File -> Open -> File -> "Yukledigimiz Dosya" -> Open" yaparak doyayi aciyoruz.
 5- F5 tusuna basarak direkt programi calistirabilirsinz
 NOT: Kodlardaki yesil yazilari okuyarak kodlarin islevini ogrenebilir ve istediginiz gibi degistirebilirsiniz
